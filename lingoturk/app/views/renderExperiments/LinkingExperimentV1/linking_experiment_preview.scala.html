Instructions to workers
Please go through all the 6 examples below to understand the task 
You will be presented with two descriptions of an everyday activity e.g. eating in a fast food restaurant. Each description consists of a sequence of sentences depicting events which are part of the activity.
Some events in the first description will be highlighted. Please choose one of them and link it to the most similar event in the second description.
For example, "order the meal" in the first description is most similar to "order meal option with server" from the second description.
Do this for each of the highlighted events. 
If you do not find a similar event in the second description, please indicate at which point the highlighted event would typically take place in the second description. For example, "wait for meal" typically occurs after "order meal option with server" and before "receive meal from server".
If you do not find a similar event in the second description or a position in the second description where the highlighted event would typically take place, please link it with "no linking possible".
For example, "server brings change" typically happens after one pays, but there is no position before or after "place credit card in holder", "server will pick up payment" and "server brings charge slip" where it would take place.
Sometimes, the highlighted sentence could depict more than one event. If you think that this it the case, please link it with all the similar events in the second description. For example, "tip the waiter and leave restaurant" depicts two events and it is linked to both "leave tip" and "leave restaurant".
Sometimes, the highlighted sentence could depict more than one event, but not all of them may be mentioned in the second description. If you think that this it the case, please link the highlighted sentence to all the mentioned events. For example, "eat and drink" depicts two events but only one of them "eat meal" is mentioned in the second description.
Sometimes, the highlighted event in the first description could correspond to more than one event in the second description.

If you think that this it the case, please link it with all the corresponding events. For example, the event "pay the bill" in the first description could include "place credit card in holder", "server will pickup payment", "server brings charge slip", "sign slip" in the second description.
